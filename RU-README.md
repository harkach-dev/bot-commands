# Команды для бота в комнате [harkach](https://wavez.fm/~/harkach)

| Команда   | Описание                                                       | Пример                           |
| :-------- | :------------------------------------------------------------- | :------------------------------- |
| !play     | Подсказывает, какой поставить трек                             | также !track или !genre          |
| !trivia   | Викторина (добавь слово answer чтобы увидеть ответ)            | !trivia <answer>                 |
| !fact     | Интересный факт                                                |                                  |
| !fortune  | Предсказание                                                   |                                  |
| !story    | Выдает случайную фразу о несуществующем жанре                  |                                  |
| !cookie   | Дать кому-то печеньку                                          | !cookie [user]                   |
| !hug      | Обнять анона                                                   | !hug [user]                      |
| !roast    | Прожарить кого-то                                              | !roast [user]                    |
| !joke     | (Несмешная) шутка                                             |                                  |
| !roll     | Генератор случайных чисел                                      | !roll [min] [max]                |
| !rps      | Камень, ножницы, бумага                                        | !rps <rock \| paper \| scissors> |
| !8ball    | Задать вопрос на да/нет                                        | !8ball <question>                |
| !coin     | Подбросить монетку                                             |                                  |
| !dice     | Бросить N-сторонний кубик                                      | !dice [сторон]                   |
| !topdj    | Рейтинг диджеев по количеству отыгранных треков                |                                  |
| !topsongs | Рейтинг песен по вутам                                         |                                  |
| !topwoot  | Рейтинг по данным вутам                                        |                                  |
| !ship     | Ship two people                                                | !ship [user1] [user2]            |
| !slap     | Шлепнуть человека                                              | !slap [user]                     |
| !thor     | Попытаться поднять молот Тора, чтобы взойти на вершину очереди | !thor                            |
| !join     | Присоединиться к рулетке                                       |                                  |
| !leave    | Покинуть рулетку                                               |                                  |

#### Только для диджеев и выше:

| Команда   | Описание                        | Usage           |
| :-------- | :------------------------------ | --------------- |
| !weather  | Показать погоду города          | !weather [city] |
| !roulette | Запустить рулетку               |                 |
| !link     | Показать ссылку на текущий трек |                 |
| !mediaid  | Показать ID текущего трека      |                 |
